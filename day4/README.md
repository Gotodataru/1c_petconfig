## Проблемы и решения

### Ошибка «Поле объекта недоступно для записи (ВидДвижения)»

В коде проведения пытался явно задать `ВидДвижения = Приход` — получал ошибку платформы.

**Решение:** убрать установку `ВидДвижения` из кода. Для оборотного регистра с типом записи «Регистратор» платформа 1С подставляет значение автоматически.
