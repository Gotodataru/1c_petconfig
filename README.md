# Учёт продаж на 1С

Учебный проект — делаю конфигурацию с нуля на платформе 1С:Предприятие 8.3. Пишу сюда то, что изучаю каждый день: что именно сделал, с чем столкнулся и как в итоге решил.

До четвёртого дня задачи были совсем простыми — просто знакомился с конфигуратором, создавал объекты через мастер. Скриншотов не делал и ничего не записывал, репозиторий решил завести только начиная с четвёртого дня.

## Что реализовано

Конфигурация для базового учёта продаж. Есть справочники Товары, Контрагенты, Склады, документ ПродажаТовара с табличной частью, регистр накопления для хранения движений и отчёт через систему компоновки данных.

## Структура репозитория

- день 4 — справочники, документ, регистр накопления, код проведения
- день 5 — переработан код проведения, запросы в консоли запросов
- день 6 — отчёт через СКД с группировками и фильтрацией по периоду
- день 7 — доработки конфигурации, добавление нового контрагента

## Что изучил

Создание справочников и документов с нуля. Регистры накопления, запись движений при проведении. Язык запросов 1С: группировки, агрегатные функции, разница между базовой таблицей регистра и виртуальной таблицей Обороты. Система компоновки данных: наборы данных, ресурсы, параметры периода, структура вывода.

## Планы

Документ ЗаказПокупателя. Отчёт по продажам с разбивкой по месяцам. Двигаюсь в сторону подготовки к сертификации 1С:Профессионал.
