# Система учёта продаж (1C:Предприятие 8.3)

> Учебный проект — разработка конфигурации с нуля, шаг за шагом

## О проекте

Практика создания собственной конфигурации на платформе 1С:Предприятие 8.3.  
Конфигурация реализует базовый учёт продаж: справочники, документы, регистры накопления, запросы.

> **Примечание:** До 4-го дня задачи были очень простыми (знакомство с конфигуратором, создание базовых объектов). Скриншотов не делал и заметок не вёл — репозиторий решил завести только с 4-го дня.

## Что изучено

**Метаданные и объекты**
- Создание справочников: Товары, Контрагенты, Склады
- Документ с табличной частью
- Регистры накопления (тип «Обороты»)

**Программирование**
- Клиентский код: автоматический расчёт суммы (событие `ПриИзменении`)
- Модуль проведения документа, запись движений в регистр
- Язык запросов 1С, группировка, агрегатные функции
- Работа с консолью запросов

**Отладка**
- Ошибка `ВидДвижения` — платформа подставляет значение автоматически для оборотных регистров
- Ошибка «Поле не найдено» в запросе — отличие базовой таблицы регистра от виртуальной `.Обороты`
