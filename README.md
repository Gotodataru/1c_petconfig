🔥 Проблема 2: Ошибка "Поле объекта недоступно для записи (ВидДвижения)"
Симптом:
При попытке указать вид движения в коде:
1c
1
Появляется ошибка: Поле объекта недоступно для записи (ВидДвижения)
Почему возникла:
В учебной версии 1С (и некоторых конфигурациях) свойство ВидДвижения может быть заблокировано для прямой записи в коде проведения.
Попытки решения:
❌ Пытались установить ВидДвижения для каждой строки → ошибка
❌ Пытались установить до цикла → ошибка
❌ Искали свойство в метаданных → не нашли
✅ Рабочее решение:
Просто убрать установку ВидДвижения из кода!
Для оборотного регистра с типом записи "Регистратор" платформа 1С автоматически подставляет ВидДвижения = Приход при проведении документа.
Итоговый код (без ВидДвижения):
1c
123456789101112131415
Вывод: Не все свойства платформы требуют явной установки. Некоторые 1С подставляет автоматически в зависимости от контекста.
