## Проблемы и решения

### Ошибка «Поле не найдено» в запросе

При обращении к виртуальной таблице `.Обороты` поля называются иначе: `КоличествоОборот`, `СуммаОборот`.

**Решение:** для простой выборки с группировкой использовать базовую таблицу регистра вместо виртуальной.
