## Проблема с запросом

При обращении к виртуальной таблице регистра через Обороты поля называются иначе — КоличествоОборот, СуммаОборот. Получил ошибку «Поле не найдено», пока не разобрался в чём дело. Решение простое: для выборки с группировкой лучше обращаться напрямую к базовой таблице регистра, а не к виртуальной.
